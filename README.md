# 📚 Book Club Management System

A web-based application to manage books, club members, borrowing logs, and event scheduling for book reading sessions.

---

## 🔧 Features

- 📘 Add, update, and delete book records  
- 👥 Manage book club members  
- 🔁 Track borrow and return of books  
- 📅 Organize and view reading events  
- 📊 Generate reports on activity

---

## 💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (FastAPI)  
- **Database:** MySQL  
- **Tools:** Git, GitHub, Razorpay, Google Maps API
